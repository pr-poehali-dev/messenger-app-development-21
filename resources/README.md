# Иконки и сплеш для Android-приложения

Положи сюда:
- `icon.png` — 1024×1024 px, квадрат, **без прозрачности**. Это базовая иконка приложения.
- `icon-foreground.png` — 1024×1024 px, **с прозрачным фоном**. Это «передний слой» адаптивной иконки Android. Сам логотип займёт примерно центральные 60% изображения, остальное — пустота.
- `icon-background.png` — 1024×1024 px, **сплошной цвет фона** под адаптивной иконкой (или текстура без важных деталей).
- `splash.png` — 2732×2732 px, центр в центре (Android обрезает по краям). Используется как сплеш-экран при старте.
- `splash-dark.png` — то же самое для тёмной темы (опционально).

## Генерация всех размеров

После добавления файлов запусти:

```bash
npx @capacitor/assets generate --android
```

Команда автоматически создаст все нужные размеры в `android/app/src/main/res/`
(mipmap-mdpi..xxxhdpi, drawable-port-*, drawable-land-*).
